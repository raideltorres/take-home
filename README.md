# Take Home

## Setup

1. Install PNPM if needed `npm install -g pnpm`
2. Run `pnpm install`
3. Run `pnpm dev`
4. Visit http://127.0.0.1:5173/

## Tools

1. PNPM: `https://pnpm.io/`
2. Vite: `https://vitejs.dev/`
3. Redux Toolkit: `https://redux-toolkit.js.org/`
